# Soybean Oil Production Analysis (1980–2021)
Data analysis and forecasting of soybean oil production using 42 years of data (1980–2021) across 5 Asian countries, completed during a Data Analyst internship.

## Overview
This project analyzes soybean oil production to identify trends, variability, and future production patterns.

## Countries Covered
- China
- India
- Japan
- South Korea
- Malaysia

## Tools & Techniques
- R (RStudio)
- Descriptive Statistics
- Time Series Analysis
- Linear Regression Forecasting

## Analysis Performed
- Computed key statistical measures (mean, median, variance, skewness, kurtosis)
- Visualized long-term production trends using time-series plots
- Built regression models to forecast production for 2022–2025
- Compared cumulative production across countries

## Project Context
Completed during a **Data Analyst Internship** at Anna Leela College of Commerce and Economics.
